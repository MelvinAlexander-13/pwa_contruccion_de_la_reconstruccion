# 🌐 Analizador de Conexión a Internet (PWA)

¡Bienvenido a **NetCheck PWA**! Esta es una Aplicación Web Progresiva (PWA) diseñada para monitorear, analizar y reportar el estado de tu conexión a internet en tiempo real. Al ser una PWA, funciona de manera fluida, es instalable y responde de inmediato incluso si te quedas sin red.

## 🚀 Características Principales

*   **Detección en Tiempo Real:** Cambios instantáneos cuando pasas de estar en línea a desconectado.
*   **Métricas de Red:** Muestra el tipo de conexión (4G, 3G, etc.) y la latencia estimada (RTT).
*   **Modo Offline:** Gracias al uso de Service Workers, la app carga y funciona sin señal.
*   **Instalable:** Añádela a la pantalla de inicio de tu celular o como app de escritorio.
*   **Ligera y Rápida:** Código limpio, sin librerías pesadas y con carga optimizada.

## 🛠️ Tecnologías Utilizadas

*   **HTML5 & CSS3:** Estructura limpia y diseño responsivo adaptado a móviles.
*   **JavaScript (Vanilla):** Lógica del estado de red usando la API `navigator.onLine` y `Network Information API`.
*   **Service Workers:** Gestión de caché para soporte sin conexión.
*   **Web App Manifest:** Configuración para habilitar la instalación en dispositivos.

## 📦 Instalación y Uso Local

Si quieres clonar este proyecto y probarlo en tu máquina local, sigue estos pasos:

1. Clona el repositorio:
   ```bash
   git clone https://github.com
   ```
2. Entra a la carpeta del proyecto:
   ```bash
   cd TU_REPOSITORIO
   ```
3. Abre el archivo `index.html` en tu navegador (o usa una extensión como *Live Server* en VS Code para que las funciones de la PWA se activen correctamente).
